  1. Why might you use the <br> tag within a paragraph, and how does it differ from the <hr> tag?
    • The <br> tag creates a new line in the code.
    • Where as the <hr> tag creates a line that croses the whole page dividing vertically anything that came before the tag with what comes after.
  2. Explain the purpose of the <div> and <span> tags in HTML, and provide a scenario where you might use each to enhance code organization.
    • The <div> tag helps divide the code into different section, defaulting to vertical separations between the sections.
    • The <span> tag also divides the code into different sections, but defaults to keeping the separations 'inline', or doesn't divide the different sections vertically.
  3. Differentiate between the class and id attributes in HTML. Provide an example scenario where using both attributes could be beneficial in organizing and styling a webpage.
    • The class attribute allows for giving a name to a section of code that you can use to identify and group any section that has the same class name.
    • Similarly the id attribute allows for giving a name to a section but the id must be a unique name that the following section will be paired to as a key-value pair.
